# My Hyprland Desktop

Моя личная сборка Arch Linux + Hyprland.

### Основные компоненты:
* **WM:** Hyprland
* **Панель:** Waybar
* **Терминал:** Kitty / Ghostty
* **Обои:** Waypaper
* **Шелл:** Fish

### Как установить на новый ПК:
1. `git clone https://github.com ~/dotfiles`
2. `chmod +x ~/dotfiles/install.sh`
3. `./~/dotfiles/install.sh`
